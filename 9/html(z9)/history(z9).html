История C++

Первые версии языка C++ появились в начале 1980-х годов благодаря датскому программисту Бьёрну Страуструпу. Он вдохновился возможностями Simula, одного из первых языков с объектно-ориентированными элементами, и решил добавить нечто подобное в C — низкоуровневый язык, в котором мало готовых инструментов.

Так появился C с дополнениями, главным из которых стали классы. В 1983 году язык C++ получил окончательное название. «++» означает «добавить единицу»: к возможностям языка программирования C добавили ещё одну.
